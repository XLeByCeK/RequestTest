# Elfen Lied - Интернет-магазин дизайнерской мебели

Проект представляет собой фронтенд-часть интернет-магазина дизайнерской мебели. 

## Функциональность

- Адаптивный дизайн
- Слайдер баннеров
- Категории товаров с возможностью раскрытия
- Анимация добавления в корзину
- Избранные товары
- Блог со слайдером
- Форма обратной связи с валидацией
- Интеграция с Яндекс.Картами

## Технологии

- HTML5
- CSS3
- JavaScript (ES6+)
- Яндекс.Карты API

## Установка и запуск

1. Клонируйте репозиторий:
```bash
git clone https://github.com/your-username/elfen-lied.git
```

2. Перейдите в директорию проекта:
```bash
cd elfen-lied
```

3. Откройте файл `index.html` в браузере или запустите через локальный сервер.

## Настройка

1. Получите API ключ для Яндекс.Карт на [https://developer.tech.yandex.ru/](https://developer.tech.yandex.ru/)
2. Замените `ваш_API_ключ` в файле `index.html` на полученный ключ:
```html
<script src="https://api-maps.yandex.ru/2.1/?apikey=ваш_API_ключ&lang=ru_RU" type="text/javascript"></script>
```
